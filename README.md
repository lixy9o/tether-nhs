# Tether — Clinical Handover Intelligence

A real-time NHS hospital-to-GP care transition
coordination platform.

## What it does
- Tracks every patient across the hospital–GP
  boundary in real time
- Assigns named ownership to every pending
  test result
- Forces GP acknowledgement of every
  discharge task
- Generates AI-assisted discharge summaries
  using Claude
- Syncs structured patient data via NHS FHIR R4

## Built with
HTML · CSS · Vanilla JavaScript ·
Anthropic Claude API · Node.js · Express

UoN MedTech Hackathon '26 — continued as a personal project
